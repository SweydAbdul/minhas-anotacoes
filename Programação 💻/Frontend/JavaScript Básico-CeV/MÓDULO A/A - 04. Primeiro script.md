# Primeiros comandos

`window.alert('msg')` -\> mensagem, com o botão **OK.**

`window.confirm('msg')` -\> mensagem, com os botões **OK.** e **Cancel**.

`window.prompt('msg')` -\> mensagem, com o campo de inserção de dados.
